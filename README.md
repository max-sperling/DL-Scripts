# DL-Scripts

## Installation

### Install package

<pre>
pip install -e <path/to/dl-scripts>
</pre>

### Uninstall package

<pre>
pip uninstall dl-scripts
</pre>

## Execution

### Playlist-Downloader

<pre>
$ download_playlist -p "https://Link/to/playlist.m3u8" -o "Video.mkv"
</pre>

### Pattern-Downloader

<pre>
$ download_pattern -u "https://Link/to/webpage.html" -p "Pattern"
</pre>
