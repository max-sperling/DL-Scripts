# DL-Scripts

## Installation

### Install package

<pre>
$ pip install -e &lt;path/to/dl-scripts&gt;
</pre>

### Uninstall package

<pre>
$ pip uninstall dl-scripts
</pre>

## Execution

### Playlist-Downloader

<pre>
$ playlist_downloader -p "https://Link/to/playlist.m3u8" -o "Video.mkv"
</pre>

### Pattern-Downloader

<pre>
$ pattern_downloader -u "https://Link/to/webpage.html" -p "Pattern"
</pre>
