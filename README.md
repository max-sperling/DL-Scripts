# DL-Scripts

## Installation

### Install package

<pre>
$ pip install -e &lt;path/to/dl-scripts&gt;
</pre>

### Uninstall package

<pre>
$ pip uninstall dl-scripts
</pre>

## Execution

### Playlist-Downloader

<pre>
$ playlist_downloader -P "https://Link/to/playlist.m3u8" -O "Video.mkv" [-H "{'Label': 'Value'}"]
</pre>

### Pattern-Downloader

<pre>
$ pattern_downloader -U "https://Link/to/webpage.html" -P "Pattern"
</pre>
